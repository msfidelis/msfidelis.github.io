O que é HTML?

  • O HTML não é uma linguagem de programação
  • O HTML é uma linguagem de Marcação de Hipertexto ou "Hypertext Markup Language"
  • É o padrão utilizado para criar estruturas estáticas de páginas da web
  • Sintaxe de caráter semântica utilizada intepretada pelos navegadores ao renderizar o DOM

Essa parada é nova?

Versão Ano

Como funciona?

  • Utilizamos tags para criar e manipular quais elementos a página irá exibir
  • Normalmente as marcações são descritas por abertura e fechamento de tags


    • <title>Titulo do Site</title>

    <p>Um parágrafo aqui</p>
